# Big Mart Sales Prediction

This project predicts product sales for Big Mart stores using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Random Forest Regressor

## Project Steps
1. Data Cleaning
2. Handling Missing Values
3. Encoding Categorical Variables
4. Model Training
5. Prediction Generation

## Dataset
The dataset contains product and outlet information such as:
- Item Weight
- Item Type
- Outlet Type
- MRP

The model predicts **Item_Outlet_Sales**.

## Output
The final predictions are stored in:
final_submission.csv
